## 핵심 기술 선택

- **SSE (EventSource)**: WebSocket 대신 단방향 스트리밍 + 방화벽 친화적
- **stdio JSON-RPC**: ACP 에이전트와 프로세스 간 통신
- **React 19 + Vite**: 빠른 HMR, 최신 React
- **Docker Compose**: agent + ui 두 서비스 동시 실행

## 다음 단계

1. `cd agent && npm install && npm test` — 유닛 테스트 실행
2. `cd agent && npm run dev` — 에이전트 서버 시작
3. `cd ui && npm install && npm run dev` — UI 시작
4. `docker-compose up --build` — 전체 스택 Docker 실행
5. ANTHROPIC_API_KEY 환경변수 설정 후 실제 ACP 에이전트 연결 테스트
