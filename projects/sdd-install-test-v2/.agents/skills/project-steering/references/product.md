## 제품 목적

개인 메모를 빠르게 작성·관리할 수 있는 CLI/API 기반 메모 앱. 단순함과 속도가 핵심 가치.

## 핵심 기능

- 메모 생성 (제목 + 본문)
- 메모 목록 조회
- 메모 상세 조회
- 메모 수정
- 메모 삭제

## 타겟 사용자

개발자 및 터미널 환경에서 빠르게 메모를 남기고 싶은 사용자.

## 핵심 제약

- 외부 데이터베이스 없이 로컬 파일 기반 저장
- CRUD 완전성 보장 (모든 기본 연산 지원)

## 범위 외

- 사용자 인증/권한 관리
- 실시간 동기화
- 웹 UI
