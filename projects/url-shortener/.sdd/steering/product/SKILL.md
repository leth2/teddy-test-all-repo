## 제품 목적
긴 URL을 짧은 코드로 단축하고, 클릭 통계를 확인하는 서비스.

## 핵심 기능
- URL 단축 (긴 URL → 짧은 코드)
- 리다이렉트 (짧은 코드 → 원본 URL)
- 클릭 통계 (총 클릭 수, 날짜별 클릭)
- 태그 분류 (URL에 태그 붙이기)

## 타겟 사용자
개발자 및 마케터

## 핵심 제약
- 짧은 코드 충돌 없음 (nanoid 사용)
- 통계 실시간 반영
- 단순하고 빠른 API
