# 2026-03-14 devlog

## cc-proxy — Claude Code OAuth → Anthropic API 범용 프록시 완성 🎉

### 배경

cline `session/prompt`가 응답 없던 문제의 근원: API 키 타입 불일치
- `sk-ant-oat01-` = Claude Code OAuth Access Token
- `sk-ant-api03-` = 표준 Anthropic REST API 키 (cline 필요)

→ OpenClaw 소스(`@mariozechner/pi-ai` v0.57.1) 분석으로 해결책 발견!

### 핵심 발견

OAuth 토큰으로 Anthropic API 호출하는 비밀:

```
Authorization: Bearer sk-ant-oat01-...
anthropic-beta: claude-code-20250219,oauth-2025-04-20,...
user-agent: claude-cli/2.1.62
x-app: cli
system: "You are Claude Code, Anthropic's official CLI for Claude."
```

`x-api-key` 방식은 "OAuth authentication is currently not supported" 에러.
Bearer + 특별 헤더 3개 + system 주입 = 성공!

### cc-proxy 구현

```
클라이언트 (x-api-key: anything, ANTHROPIC_BASE_URL=...)
  ↓
cc-proxy :8787 (0.0.0.0 바인드)
  x-api-key 무시, CLAUDE_CODE_TOKEN 환경변수로 Bearer 변환
  ↓
api.anthropic.com/v1/messages (OAuth 방식)
```

- TypeScript + Express + `@anthropic-ai/sdk`
- stream true/false 모두 지원
